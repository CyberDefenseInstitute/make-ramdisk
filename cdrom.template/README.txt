Live CD-Rom usage
-----------------

You can either boot from this CD or run it using vmware.

In order to run from vmware, you need to download VMWare Player (free) from
http://www.vmware.com/

After installing the VMWare software, copy vmware folder from this CD to your
desktop and then doubleclick on the livecd.vmx file inside the vmware folder on
your desktop.

When running under vmware, you might want to turn Ethernet into Bridged mode,
but in this case Windows Firewall might block packets.


このCDを使用するには、CDから直接起動するか、VMWareを使用してください。

VMWareの場合は、以下のURLからVMware Player(無料)をダウンロードしてください。
http://www.vmware.com/

VMWareをインストール後、このCDからvmwareフォルダをハードディスクにコピーし、
その中に入ってるlivecd.vmxファイルをダブルクリックしてください。
